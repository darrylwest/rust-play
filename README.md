## Utilities

_These utilities install with cargo install..._

* tldr - to replace man
* lsd - file lister
* dust - disk use, install with `cargo install du-dust`


## Cargo

* `cargo new --vcs git -v <path>`
* `cargo run`
* `cargo test`
* `cargo build --release`
* `cargo clean`
* `cargo instal <app>`
* `cargo doc --open`
